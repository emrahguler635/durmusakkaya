# Public Klasörü - Resimler

Bu klasöre eklediğiniz tüm dosyalar web sitesinde kullanılabilir.

## Resim Ekleme

1. Resim dosyasını bu klasöre kopyalayın (örn: `haber1.jpg`)
2. Yönetim panelinden haber eklerken, "Görsel URL" alanına şunu yazın:
   - Production: `/durmusakkaya/haber1.jpg`
   - Development: `/haber1.jpg`

## Mevcut Resimler

- `logo.png` - Site logosu
- `profile.jpg` - Profil resmi
- `og-image.png` - Varsayılan haber resmi
- `slider1.jpg` - Slider resmi
- `favicon.svg` - Site ikonu

## Önerilen Resim Formatları

- **Format**: JPG veya PNG
- **Boyut**: Haberler için 1200x630px (16:9 oranı)
- **Dosya boyutu**: Mümkün olduğunca küçük (max 500KB)

## Harici URL Kullanımı

Eğer resimleri başka bir serviste (Imgur, Cloudinary, vs) barındırıyorsanız:
- Tam URL'yi kullanın: `https://example.com/resim.jpg`
- Yönetim panelinde "Görsel URL" alanına direkt yapıştırın
